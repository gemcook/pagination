## Installation

```
$ npm install --save @gemcook/pagination
```

or

```
yarn add @gemcook/pagination
```

## Usage

```
<Pagination
  current={store.state.current}
  total={100}
  changePage={current => {
    store.set({current});
  }}
/>
```
